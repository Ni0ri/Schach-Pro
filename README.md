# ♟ Schach Pro – Android App

Kivy-basiertes Schachspiel für Android mit eingebauter KI.

## Features
- KI vs KI, Mensch vs KI, Mensch vs Mensch
- KI-Stärke einstellbar (1–3)
- Undo-Funktion
- Piece-Square-Tables für intelligente KI
- Schachmatt, Patt, 50-Züge-Regel, Stellungswiederholung

## APK bauen
Die APK wird automatisch via GitHub Actions gebaut:
1. Code auf `main` pushen
2. Unter **Actions** den Build abwarten (~20 Min)
3. Unter **Artifacts** die APK herunterladen
